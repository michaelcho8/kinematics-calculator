Kinematic Equations — SUVAT Reference

These four equations relate the five 1D kinematic variables: initial velocity (u), final velocity (v), acceleration (a), displacement (s), and time (t). If you know any three, you can solve for the other two.

v = u + atFinal velocity from initial velocity, acceleration, and time
s = ut + ½at²Displacement when acceleration is constant
v² = u² + 2asRelates velocity and displacement without time
s = ½(u + v)tAverage velocity × time (constant acceleration only)

2D motion — independent x and y components

2D motion treats horizontal and vertical components separately. Each component obeys the same 1D equations, sharing the time t:

  • vₓ = uₓ + aₓt, vᵧ = uᵧ + aᵧt — final velocity components
  • sₓ = uₓt + ½aₓt², sᵧ = uᵧt + ½aᵧt² — displacement components
  • |v| = √(vₓ² + vᵧ²), angle = atan2(vᵧ, vₓ) — magnitude and direction

Projectile motion is a special case of 2D

Projectile motion has no horizontal acceleration (aₓ = 0) and constant downward gravity (aᵧ = −g). The calculator decomposes the launch into components:

  • vₓ = v₀ cos θ — horizontal velocity (constant)
  • vᵧ = v₀ sin θ — initial vertical velocity
  • T = 2vᵧ / g — total time of flight (returns to launch height)
  • H = vᵧ² / 2g — maximum height
  • R = vₓ · T — horizontal range
